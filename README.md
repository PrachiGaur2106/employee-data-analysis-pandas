# Employee Data Analysis using Pandas

This is a beginner-level data analysis project using Python and Pandas.

## Project Overview
In this project, I created an employee dataset and performed basic data analysis.

## Tasks Performed
- Created DataFrame
- Viewed summary using describe()
- Used groupby() function
- Filtered data using conditions
- Exported dataset to CSV file

## Tools Used
- Python
- Pandas
- Jupyter Notebook
